# NetTyan-3D-generator

## Описание проекта
NetTyan-3D-generator — это проект, направленный на создание 3D-моделей с использованием машинного обучения. Цель проекта — автоматизировать процесс генерации 3D-объектов для использования в играх, анимации и других областях.
